API & Data

This page documents the data sources used in the demo and the third-party libraries integrated client-side. The demo is client-first; for production you should add server endpoints to persist bookings, provider records, and to generate signed/secure exports. Also restrict any public API keys by HTTP referrer.

Data Sources

  • USDA/NASS — Estimated Annual Agricultural Pesticide Use (2013–2017) (pre-aggregated into data/pesticide_summary.json)
  • NRC — Radioisotope Facility Locations Database (used as a probe for irradiation provider locations)
  • Local demo data — providers.csv (seeded sample providers for the map)

External APIs & Libraries

  • Google Maps JavaScript API (Places) — provider maps & address autocomplete (restrict keys by HTTP referrer)
  • Chart.js (CDN) — interactive visualizations
  • jsPDF (CDN) — client-side PDF export for quick demos
  • Tailwind CSS (CDN) — utility styles used in some pages
